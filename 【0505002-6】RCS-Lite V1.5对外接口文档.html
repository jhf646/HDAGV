取消任务

基本信息

Path： /rcms/services/rest/hikRpcService/cancelTask

Method： POST

接口描述：

功能说明:通过正在执行的任务编号,取消该任务,不再执行，潜伏车如果背着货架,会把货架直接放在路上；叉车背着容器，会把容器放在路上，同时小车状态置为空闲状态；

请求:
{
    "reqCode": "1541954B96B1112",
    "reqTime": "",
    "clientCode": "",
    "tokenCode": "",
    "forceCancel": "1",
    "matterArea": "abc",
    "agvCode": "",
    "taskCode": "123456"
}
应答:
{
    "code": 0,
    "message": "成功",
    "reqCode": "B6A831705D9CD58C"
}

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/json 是

Body

名称类型是否必须默认值备注其他信息
reqCodestring必须请求编号
reqTimestring非必须请求时间
clientCodestring非必须
tokenCodestring非必须
forceCancelstring非必须取消类型，0表示取消后货架直接放地上，1表示agv仍然背着货架，根据会哭区域执行回库指令，只有潜伏车支持，若不传值，默认的取消模式为0.
matterAreastring非必须回库库区编号，forceCancel=1时有效
agvCodestring非必须小车编号。agvCode和taskCode不能同时为空
taskCodestring非必须任务编号，agvCode和taskCode不能同时为空

}
